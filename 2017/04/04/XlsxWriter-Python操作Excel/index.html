发表于 2017-04-04 | 分类于 Python

XlsxWriter

xlsxwriter是创建excel文件的python模块，适用2007 + 版本，支持的功能：

  • 完全兼容Excel格式文件[compatible Excel files]
  • 格式化[full formatting]
  • 单元格合并[merged cells]
  • 命名[defined names]
  • 图表[charts]
  • 自动筛选[autofilters]
  • 数据验证和下拉列表[data validation and drop down list]
  • 条件格式[conditional formattingt]
  • worksheet PNG/JPEG images
  • Rich multi-format strings
  • cell comments
  • textboxes
  • Pandas接口
  • 内存优化模式写入大文件[memory optimization mode for writing large files]

XlsxWriter简介

xlsxwriter在写Excel文件方面的优缺点：

  • 优点：高度兼容excel并友好支持excel特性；可配置以使用较小的内存进行快速写入较大的文件。
  • 缺点：无法读取或修改已存在的excel文件。

安装

两大核心对象：WorkBook & WorkSheet

强调：只能创建新的excel文件，无法读取或修改已存在文件；写入数据是在worksheet对象中操作的，并且在worksheet中A1的索引是(0,0)。

三大核心方法：文件创建 & 数据写入 & 格式化

数据写入主要通过worksheet.write(row,col,token,[format])方法，格式化主要通过workbook.add_format()方法。
